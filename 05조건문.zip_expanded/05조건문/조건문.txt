

조건문(conditional statements)
조건문은 주어진 조건식의 결과에 따라 별도의 명령을 수행하도록 제어하는 명령문입니다.
조건문 중에서도 가장 기본이 되는 명령문은 바로 if 문입니다.
---------------------------------------
 
자바에서 사용하는 대표적인 조건문의 형태는 다음과 같습니다.
1. if 문
2. if   else 문
3. if    else if     else 문
4. switch 문

--------------------------------------

if 문

if 문은 조건식의 결과가 참(true)이면 주어진 명령문을 실행하며, 
거짓(false)이면 아무것도 실행하지 않습니다.

---
 
if 문 문법

if (조건식) {
    문장1; //조건식에 만족할 경우 실행되는 문장
}
다음 문장;


설명 : if문에서 조건식에 만족할 경우에, 즉 조건식의 결과가 참일 때 문장1이 수행되고 다음문장이 수행됩니다.
만일 조건식에 만족하지 않을 경우, 즉 조건식의 결과가 거짓일 때 문장1의 수행되지 않고 곧바로
다음 문장이 수행됩니다.


>예제 절대값 구하기 if01.java

---------------------------------------


if  else 문

if 문과 함께 사용하는 else 문은 if 문과는 반대로 
주어진 조건식의 결과가 거짓(false)이면 주어진 명령문을 실행합니다.

두 가지 경우 중에 하나가 실행되도록 하려면 if else 문을 사용합니다.

---

if  else문 문법

if(조건식){
	문장1; //조건식에 만족할 경우 실행되는 문장
}else{
	문장2; //조건식에 만족하지 않을 경우 실행되는 문장
}

설명 : if문에서 조건식의 결과가 참일 경우에는 if문 바로 다음에 기술한 문장1이 수행됩니다.
조건식의 결과가 거짓일 때 else다음에 기술된 문장2가 수행됩니다. 


>예제 if  else 문을 사용하여, 해당 문자가 영문 소문자인지 아닌지를 확인하는 예제.   if02.java


---------------------------------------

if    else if     else 문
- 조건식을 여러 개 명시할 수 있는 구문 이다.



if    else if     else 문 문법

	if (조건식1) {
	
	    조건식1의 결과가 참일 때 실행하고자 하는 명령문;
	
	} else if (조건식2) {
	
	    조건식2의 결과가 참일 때 실행하고자 하는 명령문;
	
	} else {
	
	    조건식1의 결과도 거짓이고, 조건식2의 결과도 거짓일 때 실행하고자 하는 명령문;
	
	}

참고 : 이때 else if 문은 여러 번 나와도 상관없지만, if 문과 else 문은 단 한 번만 나올 수 있습니다.


>예제 if    else if     else문을 사용하여,  
     해당 문자가 영문 소문자나 영문 대문자인지, 아니면 영문자가 아닌지를 확인하는 예제.  if03.java



---------------------------------------

switch 문
- if  else 문과 마찬가지로 주어진 조건식의 결과에 따라 프로그램이 다른 명령을 수행하도록 하는 조건문입니다




switch 문 문법

	switch (조건 값) {
	
	    case 값1:
	
	        조건 값이 값1일 때 실행하고자 하는 명령문;
	
	        break;
	
	    case 값2:
	
	        조건 값이 값2일 때 실행하고자 하는 명령문;
	
	        break;
	
	    ...
	
	    default:
	
	        조건 값이 어떠한 case 절에도 해당하지 않을 때 실행하고자 하는 명령문;
	
	        break;
	
	}



>예제   switch 문을 사용하여, 해당 문자가 영문자 모음인지를 확인하는 예제  switch04.java
