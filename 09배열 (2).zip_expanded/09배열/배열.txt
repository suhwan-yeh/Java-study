



배열 : 동일한 자료형의 여러 개의 데이터를 저장하기 위해 여러 개의 방을 한꺼번에 만들어 사용할수 있는 메모리 공간을 말합니다. 

배열 메모리 공간을 만드는 2가지 방법(배열 을 선언하는 2가지 방법)
	방법1. new 연산자를 이용하여 배열 메모리를 만드는 방법	
	방법2. 해당 배열의 내용을 직접 초기화 하는 방법으로 배열 메모리를 만드는 방법

방법1. new 연산자를 이용하여 배열 메모리를 만드는 방법

   그림1
    //5명의 점수를 저장하기 위한 정수형 배열 메모리를 만든 예 입니다.
	int []score = new int[5];	
	
	[		]	[		]	[		]	[		]	[		]
	score[0] 	 score[1] 	score[2]	score[3]	score[4]

         
        int []score 중에 score <- 배열메모리 이름
        new int[5]; 중에 int <- 각 원소에 저장할 값에 대한 자료형
        new int[5]; 중에 [5] <- 배열의 원소의 개수
    	
    	int자료형 다음에 대괄호[]가 나오고 몇개의 데이터를 저장할지 결정 해야 하므로 배열의 원소 개수를 기술해야합니다.
   
    
   그림2
	  [	 0	]	[	0	]	[	0	]	[	0	]	[	0	]
	  score[0] 	 score[1] 	score[2]	score[3]	score[4] 
	        
	  배열은 메모리(RAM)에 여러 개의 방을 연속된 기억공간으로 할당합니다.
	 배열은 여러 개의 방들을 동일한 이름(배열명)으로 접근하도록 합니다.
	 배열에 저장된 데이터는 반복문을 이용하여 일괄처리를 할 수 있어서 데이터 처리가 편해집니다.
	 위 score배열에는 5개까지 데이터를 저장할 수 있습니다.
	 한 명의 점수(정수값)을 저장할수 있는 기억장소 하나를 배열의 요소라고 부릅니다.
	 배열의 요소를 각 각 개별적으로 접근하여 데이터를 저장하거나 읽어오기 위해서는 배열의 몇 번쨰 위치한 기억공간인지를 설명하기 위해서
	 첨자(색인,index)를 지정 해야 합니다.
	            
 	배열의 첨자는 0번부터 시작하여 번호를 1씩 증가시킵니다.
 	첫 번째 요소는 score[0], 
 	두 번째 요소는 score[1] 로 접근할 수 있습니다.
 	마지막 원소는 선언된 원소의 개수보다  하나 적은 번호 4가 됩니다.
 	5개의 점수를 저장하기 위한 배열의 마지막 원소에 접근하기 위해서는 score[4]라고 표현합니다.
 	
 	이렇게 new 연산자를 이용히여 배열을 선언(배열 메모리 만들기)하게 되면 숫자 배열인 경우에는
 	0, 불린 자료형인 경우에는 false, 문자 자료형인 경우에는 NULL문자로 초기화됩니다.

 방법2. 해당 배열의 내용을 직접 초기화 하는 방법으로 배열 메모리를 만드는 방법 
	
	배열 선언은 기억공간을 할당 해주는 역할을 합니다.
	선언된 배열에 데이터를 저장할 때에는 한꺼번에 데이터를 저장하지 못하고 
	인덱스번호(첨자)를 지정하여 원소 하나 하나에 데이터를 저장해야 합니다.
	이러한 불편함은 배열의 초기화로 햐결할 수 있습니다.
	배열을 선언하면서 값을 대입할 수 있는데, 기억공간이 할당될 때 값을 대입하는 것을 초기화라고 합니다.
         초기화 과정에서는 값을 한꺼번에 부여할 수 있습니다.

 다차원 배열
	1차원이 배열이 아닌 2, 3, 4.. 차원 배열을 다차원 배열이라고 합니다.
	2차원 배열까지는 많이 사용되지만 그 이상은 로직이 복잡해 질수 있으므로 많이 사용하지 않습니다.
	다차원배열을 이해하기 위해서 2차원 배열까지 살펴보도록 하겠습니다.
	
	그림
				국어		영어		수학
		1번학생 	85		60		70
		2번학생 	90		95		80
		3번학생 	75		80		100
		4번학생 	80		70		95
		5번학생 	100		65		80
		
	위그림에 대한 설명
		2차원 배열은 1차원 배열의 모임이라고 할 수 있습니다.
		학생 5명의 국어, 영어, 수학 점수를 처리하기 위해서는 데이터를 위의 그림과 같이
		저장할 수 있습니다.
		2차원 배열은 위 그림과 같이 행렬 형태로 데이터를 처리할 때 사용합니다.
		학생의 정보가 행과 열에 일정한 형태로 저장되어 있습니다.
		행 단위에는 각 학생의 정보가 저장되고 열 단위로는 각 과목별 점수가 저장되어 있습니다.
		
		
	  그림	  	
	  	 //3과목을 치룬 학생 5명의 데이터를 저장하기 위해서는 다음과 같이 2차원 배열을 선언해야 합니다.
	  	 int [][] score = new int [5][3];
	  	 	  	 
	  	 			0열			1열				2열
	  	 0행      score[0][0]	score[0][1]		score[0][2]		
	  	 
	  	 1행      score[1][0]	score[1][1]		score[1][2]
	  	 
	  	 2행      score[2][0]	score[2][1]		score[2][2]
	  	 
	  	 3행      score[3][0]	score[3][1]		score[3][2]
	  	 
	  	 4행      score[4][0]	score[4][1]		score[4][2]
		
	그림 설명
		int [][] score = new int [5][3];로 2차원 배열을 선언하면
		위 그림과 같은 5행 3열의 기억공간이 만들어 집니다.
		2차원 배열인 score는 15(5X3)개의 원소를 저장할 수 있도록 메모리가 할당 됩니다.
		
	그림
		score[2][1] = 80;
		
	그림 설명
		15개의 정수값을 저장할 수 있는 2차원 배열 공간 중 특정 위치에 데이터를 저장하기 위해서는
		그 위치를 행과 열의 첨자(인덱스위치번호)로 지정해야 합니다.
		3번 학생의 영어 점수를 80점으로 저장하기  위해서는 위 그림의 score[2][1] = 80;로 작성 해야합니다.
