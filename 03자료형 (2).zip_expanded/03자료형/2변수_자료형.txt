
변수?
-프로그램 실행 중에 변하는 값을 저장할 수 있는 컴퓨터 내부의 메모리 기억공간 (변수는 데이터를 담는 일종의 그릇)
-변수 메모리 공간은 앞에서 언급한 상수(값)을 저장하기 위한 컴퓨터 내부의 메모리 공간
-변수 메모리 공간은 컴퓨터의 메모리(RAM)의 특정 위치(스택)에 만들어 집니다.

예)  10 상수(값)  저장-> 변수 메모리 기억공간



변수선언?
- 변수 메모리 공간을 컴퓨터의 메모리(RAM)의 특정 위치에 만드는일



변수선언 문법

	자료형 만들변수이름;
	
  예)int	 age;		// 4바이트 크기의 정수 형태의 데이터를 저장할 age라는 이름의 변수메모리 선언
  
  중요! 변수를 선언하기 위해서는 자료형과 변수의 이름을 결정 해야 합니다.



자료형(데이터형)이란?
- 프로그래밍 언어에서 실수,정수 등 여러 종류의 데이터를 식별하는 분류로서 자료의 형태를 의미함.
- 서로 크기가 다른 데이터(상수)를  변수 메모리에 저장시! 만들 변수 메모리의 크기를 결정 하는데 사용되는 범위. 


자료형(데이터형)의 종류
1. 기본 자료형
	수치형
		정수형
			byte, char, short, int, long
		실수형
			float, double
	논리형
		boolean
			
2. 레퍼런스 형
   	클래스
   	인터페이스
   	배열
	


자료형(데이터형)의 종류 - 기본 자료형 - 수치형 - 정수형 살펴보기

	그림
		종류		정수유형		크기					허용값
				byte		1byte(8 bit)		-128 ~ 127
		정수형	short		2byte(16 bit)		-32768 ~ 32767
				int			4byte(32 bit)		-2147483648 ~ 2147483647 (약 -21억4천 ~ 21억4천 )
				long		8byte(64 bit)		-9223372036854775808 ~ 9223372036854775807
                           

	설명
	정수형은 자바의 기본 자료형 중에 가장 대표적인 정수형입니다.
	정수형을 저장할 수 있는 자료형은 크게 byte, short, int, long으로 나뉩니다.
	정수를 저장하기 위한 자료형을 4가지로 나눈 이유는 저장할 데이터(상수,값)에 따라 컴퓨터 내부의 변수메모리 기억공간의 크기를 다르게 할당 하기 위해서 입니다.

레퍼런스 형의 종류 - 클래스 - 문자열형(String) 상수 살펴보기

	문자열형(String) 상수 : 두 문자 이상을 이중 따옴표" "로 둘러싼 상수를 문자열 상수라고 합니다.
	
	예)	String y = "AB"; //문자열 "AB"를  저장하기 위한 변수y 선언시 String클래스자료형을 제공합니다.
